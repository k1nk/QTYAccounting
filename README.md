# QTYAccounting
数量簿記は、仕訳に数量やその他の経営に関する情報を付加し、経営の分析に役立てる試みです。
記述方法などの概要については、「数量簿記.md」をご覧ください。
また、実際の計算例については、jupyter notebook「introduction.ipynb」を実行してご確認ください。
